How it thinks

Motion means meaning.

The layout never moves unless the driver actually needs to look. Mode upgrades are instant; downgrades wait for six seconds of sustained calm. No dancing gauges.

Deviation, not raw values.

Oil pressure tracks RPM. Fuel pressure tracks boost. WAPOW shows how far you are from normal for right now — steady when healthy, loud when it isn't.

Sentinel gauges.

Oil pressure, fuel pressure, AFR live in a small fixed home row and promote themselves to the main view only when something's wrong. Alerts triage by severity, automatically.

Open source, built in public

WAPOW is GPL v3 — the code is free, forever. We make our money selling assembled, tested units so you don't have to solder anything.

Built by a former Formula Drift crew chief, documented on camera from the first breadboard onward. Test car: a 2000 Impreza 2.5RS with an STI heart.
